# springboot系列教程

## 环境搭建

确保计算机已装docker

代码所有所需环境可在[https://gitee.com/fz972/docker-env.git][https://gitee.com/fz972/docker-env.git]中找到

搭建步骤

```shell
git clone https://gitee.com/fz972/docker-env.git
cd docker-env
docker-compose up -d
```
